Installation Setup - MAC

  • Install Ruby - brew install ruby
  • Install Cucumber - gem install cucumber
  • Install Capybara - gem install capybara
  • Install Selenium webdriver - gem install selenium-webdriver
  • Install Rspec - gem install rspec

Installation Setup for Windows

  • Install Ruby - Ruby Installer
  • Install Cucumber - gem install cucumber
  • Install Capybara - gem install capybara
  • Install selenium webdriver - gem install selenium-webdriver
  • Install rspec - gem install rspec

Step Definitions

Given(/^I go to bookmyshow website$/) do visit "https://in.bookmyshow.com/chennai" end

Then(/^I verify the experiences link is available$/) do find("//a[contains(@href,'http://in.bookmyshow.com/chennai/experiences/')]").text.should eq("EXPERIENCES") end

Given(/^I maximize browser$/) do page.driver.browser.manage.window.maximize end

When(/^I click on the signin link$/) do find(".//*[@id='preSignIn']").click end

When(/^I enter the username as "([^"])"$/) do |username| find(".//[@id='iUserName']").set username end

When(/^I enter the password as "([^"])"$/) do |password| find(".//[@id='iPwd']").set password end

And(/^I click on the sign in button$/) do find(".//*[@id='iUserNameParent']/a/div").click end

Then(/^I verify the error text as "([^"])"$/) do |errortext| find(".//[@id='dSignInErrorPassword']/span[2]").text.should have_text(errortext) end

Then(/^I verify in the footer section of movies by language has the following languages$/) do |table| # table is a Cucumber::Core::Ast::DataTable table.hashes.each_with_index do |hash,index| find("//div[contains(@class,'movies-by-language')]//li[#{index+1}]//span").text.should eq(hash["Languages"])
end end
